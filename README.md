# TravelPlannerApp

旅行計画を整理・管理するためのWebアプリケーションです。

---

# 🧭 アプリ概要

Next.js と Supabase を用いて開発している  
旅行の予定を整理するためのシンプルなWebアプリです。

旅行前に

・どこへ行くか  
・何をするか  
・日程をどう組むか  

をメモ感覚で整理できることを目的としています。

フロントエンドとバックエンドを組み合わせた  
フルスタック開発の学習として制作しています。
