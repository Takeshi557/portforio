# TravelPlannerApp

学習用として作成しているシンプルな旅行計画アプリです。

---

# アプリ概要

Next.js と Supabase を使って開発している
**旅行の予定を整理するためのシンプルなWebアプリ**です。

旅行前に「どこ行く？何する？」をメモ感覚で整理できるアプリを目指して作成しています。

フルスタック開発の基礎を学ぶことを目的に制作しました。
